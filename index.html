5 — Storage Storage strategy ▶
localStorage (primary)
key: wl__{shopDomain} prevents multi-shop collisions guests + logged-in users
Local data schema
version: 1 items: [ProductDataModel] layoutPreference: "grid"|"list"
Customer metafield updated
single metafield, JSON array stores product IDs only product data fetched fresh on render
Debounced batch sync new
2s after last change fires batches rapid add/remove actions logged-in users only
Failed sync queue new
failed syncs queued in localStorage silent retry on next page visit no error UI shown to user
Quota + fallback
all reads/writes in try/catch fallback to sessionStorage handles private/incognito mode
Multi-tab sync
window.storage event listener re-renders UI on external change debounced to avoid thrashing
Login merge logic new
fetch metafield on login deduplicated union of local + remote write merged list back to metafield
6 — Sort / Filter Sort and filter ▶

Filter options
in stock only out of stock only sent as query params to API
Sort UI
<select> dropdown in popup header + page header triggers API call on change
Filter UI
toggle chips / checkboxes in popup header + page header triggers API call on change
